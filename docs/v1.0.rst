Introduction
------------

This project provides the identityfunction.

Installation
------------

To install identityfunction, you can use pip. Open your terminal and run:

.. code-block:: shell

    pip install identityfunction

Features
--------

``identityfunction.identityfunction(value: Any, /) -> Any``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This function returns the value given to it.

``identityfunction.main``
~~~~~~~~~~~~~~~~~~~~~~~~~

This click command implements a CLI for the identityfunction.

``identityfunction.test() -> unittest.TextTestRunner``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This project can be tested through its test function.

.. code-block:: shell

    import identityfunction
    identityfunction.test()
