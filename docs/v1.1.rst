Introduction
------------

This project provides the ``identityfunction``.

Installation
------------

To install ``identityfunction``, you can use ``pip``.
Open your terminal and run:

.. code-block:: shell

    pip install identityfunction

Typing
------

This project is strictly typed with ``mypy``.

.. container:: versionadded

    **Added in version 1.1.**

Features
--------

``identityfunction.identityfunction(value: Value, /) -> Value``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This function returns the value given to it.

``identityfunction.main``
~~~~~~~~~~~~~~~~~~~~~~~~~

This click command implements a CLI for the identityfunction.

``identityfunction.tests.test() -> unittest.TextTestResult``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This project can be tested through its test function.

.. code-block:: shell

    import identityfunction.tests
    identityfunction.tests.test()
